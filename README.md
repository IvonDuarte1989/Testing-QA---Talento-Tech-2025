# Testing QA-TalentoTech (2025)

## Contenidos principales:
* Fundamentos del testing de software y rol del QA en el ciclo de desarrollo.
* Análisis de requerimientos mediante historias de usuario y criterios de aceptación.
* Diseño y creación de casos de prueba utilizando técnicas como partición de equivalencia y valores límite.
* Ejecución de pruebas funcionales y exploratorias con registro de resultados y evidencias.
* Identificación, documentación y seguimiento de defectos, comprendiendo su ciclo de vida.
* Uso de herramientas de gestión y trabajo en entornos ágiles y colaborativos, desarrollando habilidades de análisis, documentación de pruebas y enfoque en la calidad del software. 

## Objetivos del Curso
* Aplicar de forma integral las prácticas de testing de software aprendidas
durante el curso para evaluar y asegurar la calidad de una plataforma web 
y mobile, mediante la planificación, ejecución de pruebas, gestión de defectos 
y uso de herramientas ágiles como Jira y Zephyr.
