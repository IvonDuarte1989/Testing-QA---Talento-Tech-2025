# Testing-QA-Talento-Tech-2025

Formación orientada a la calidad de software y testing manual, enfocada en 
el análisis, diseño y ejecución de pruebas para garantizar el correcto 
funcionamiento de aplicaciones.

## Contenidos principales:
## Fundamentos de testing de software
* Rol del QA dentro del ciclo de desarrollo
* Conceptos de calidad de software
* Tipos y niveles de pruebas
## Análisis de requerimientos
* Interpretación de historias de usuario y criterios de aceptación
* Identificación de escenarios de prueba
## Diseño de casos de prueba
* Creación de casos de prueba claros y estructurados
* Técnicas de diseño de pruebas (partición de equivalencia, valores límite, entre otras)
## Ejecución de pruebas
* Pruebas funcionales y exploratorias
* Registro de resultados y evidencias
## Gestión y reporte de defectos
* Identificación y documentación de bugs
* Ciclo de vida del defecto
* Uso de herramientas de seguimiento
## Metodologías de trabajo
* Introducción a metodologías ágiles
* Trabajo en entornos colaborativos

## Competencias desarrolladas:
* Análisis de requerimientos desde la perspectiva de QA
* Diseño y documentación de casos de prueba
* Ejecución de pruebas funcionales
* Reporte claro y detallado de defectos
* Pensamiento analítico y enfoque en la calidad del software

## Objetivos del Curso
Aplicar de forma integral las prácticas de testing de software aprendidas
durante el curso para evaluar y asegurar la calidad de una plataforma web 
y mobile, mediante la planificación, ejecución de pruebas, gestión de defectos 
y uso de herramientas ágiles como Jira y Zephyr.
